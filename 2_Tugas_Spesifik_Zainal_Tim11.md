## 1. Tujuan & Logika

### Mengapa Pengukuran Fisik Diperlukan?

| Masalah | Solusi |
|---------|--------|
| Warna buah Pra-Matang (M2) dan Mentah (M3) sama-sama hitam | Gunakan dimensi fisik sebagai pembeda |
| AI perlu referensi ukuran dalam gambar | Batang yang terukur di 150cm menjadi "penggaris" |
| Posisi buah mengindikasikan kematangan | Dokumentasi posisi pelepah (atas/bawah) |

### Hipotesis Biologis yang Divalidasi

```
Pohon Sawit:
┌────────────────────┐
│     Pelepah Atas   │  ← Buah MUDA (belum matang)
│          ↑         │
│          │         │
│          │         │
│     Pelepah Bawah  │  ← Buah TUA (akan matang)
│          │         │
│      [BATANG]      │  ← Titik ukur 150cm
│          │         │
└──────────┴─────────┘
        Tanah
```

## 2. Protokol Pengukuran Manual

### 2.1 Target Sampel
- **Jumlah:** 100 Pohon terpilih
- **Penentuan:** Di tempat/lapangan saat pengambilan data

### 2.2 Parameter yang Diukur

#### Parameter 1: BATANG (Trunk)

| Item | Spesifikasi |
|------|-------------|
| **Apa yang diukur** | Keliling Batang |
| **Ketinggian pengukuran** | 150 cm (1,5 meter) dari tanah |
| **Alat** | Meteran pita (karena batang melengkung) |

> **Syarat:** Saat pohon difoto, titik 150cm ini harus masuk dalam frame foto. Ini adalah kunci kalibrasi.

#### Parameter 2: BUAH SAWIT (TBS - Tandan Buah Segar)

| Item | Cara Ukur |
|------|-----------| 
| **Tinggi Buah** | Ukur vertikal dari ujung bawah ke ujung atas |
| **Keliling Buah** | Ukur horizontal/lingkar buah |
| **Posisi Pelepah** | Catat: Lingkar bawah (Tua) atau Lingkar atas (Muda) |

**Kategori Sampel Buah:**
- [ ] Buah Besar (representasi matang/pra-matang)
- [ ] Buah Sedang (representasi transisi)
- [ ] Buah Kecil/Bunga (representasi muda)

### 2.3 Alat yang Diperlukan

| Alat | Fungsi |
|------|--------|
| Meteran pita | Mengukur keliling (batang melengkung) |
| Penggaris/meteran rigid | Mengukur tinggi buah |
| Logbook/Buku catatan | Pencatatan manual |
| HP (Excel/Notes) | Backup pencatatan digital |

## 4. Workflow Pengukuran di Lapangan

```
┌─────────────────────────────────────────────────┐
│ 1. Pilih pohon sampel di lapangan               │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│ 2. Ukur keliling batang @150cm                  │
│    → Catat & mark titik untuk referensi foto    │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│ 3. Identifikasi sample buah (3 kategori)        │
│    → Atas (Muda), Tengah, Bawah (Tua)           │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│ 4. Ukur tinggi & keliling setiap sample buah    │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│ 5. Foto pohon dari 4 sisi (U-T-S-B)             │
│    → Pastikan titik 150cm masuk frame           │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│ 6. Catat semua data di logbook/HP               │
│    → Hubungkan dengan ID foto                   │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│ 7. Lanjut ke pohon berikutnya                   │
└─────────────────────────────────────────────────┘
```

## 5. Checklist Tugas Zainal

### Persiapan
- [ ] Siapkan logbook/form cetak
- [ ] Meteran pita (minimal 3 meter)
- [ ] Penggaris/meteran untuk tinggi buah
- [ ] HP dengan kamera sesuai SOP
- [ ] Backup power bank

### Di Lapangan (per pohon)
- [ ] Ukur keliling batang @150cm
- [ ] Ukur minimal 3 sample buah (posisi berbeda)
- [ ] Catat posisi pelepah setiap buah
- [ ] Foto 4 sisi dengan batang @150cm dalam frame
- [ ] Catat di logbook dengan ID foto

### Pasca Lapangan
- [ ] Input data ke Excel
- [ ] Verifikasi semua 100 pohon tercatat
- [ ] Hubungkan data ukuran dengan foto masing-masing
